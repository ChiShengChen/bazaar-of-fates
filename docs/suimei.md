# 四柱推命（日系）圖解 · Shichū-Suimei Visual Guide

和八字一樣的四柱干支，但用**日本京都泰山流**的讀法：主看 **十二運星** 與 **天中殺（空亡）**。
Same four 干支 pillars as BaZi, but read the **Japanese** way — the spine is the 十二運星 life-stage cycle and 天中殺 (the void pair).

> 開啟 / Open: 首頁選 **Shichū-Suimei · 四柱推命（日）**。

## 命盤怎麼讀 / Reading the chart

```
 年柱   月柱   日柱   時柱
 庚午   壬午   辛亥   乙未
 (十二運星：帝旺 / 病 / 長生 …)        ← 日主對各支的生命階段
 (藏干：本氣→餘氣)
 日主 辛(金) · 天中殺(空亡)：寅卯
```

- **十二運星**：日主對每個地支的「生命力階段」（長生→帝旺強旺，病死墓絕衰弱）。
- **天中殺（空亡）**：由日柱旬推出的一對「空」地支——傳統說在這兩個流年宜低調蟄伏。
- **藏干**：每個地支裡藏的天干（本氣＋餘氣）。
