# 鐵板神數圖解 · Iron Plate Visual Guide

數術一派：由出生日期「起命數」，得一個**命數**，再由流年數對應斷語。
A numerological system: the birth date yields a **命數 (life number)**; each year's number maps to a verdict.

> 開啟 / Open: 首頁選 **Iron Plate · 鐵板神數**。只需日期。

## 怎麼讀 / Reading it

```
 命數 52
 流年 (verse_fortune)：吉 / 平 / 凶
```

- **命數**：一生的基準數（由干支起數法推出）。
- **流年數**：把命數與當年干支組合，得到該年的條目斷語。
