# 奇門遁甲圖解 · Qi Men Dun Jia Visual Guide

「三式」之一：把時間排成**九宮格 + 八門**的局，看當下的格局與值符門。
One of the Three Cosmic Boards (三式): time is laid into a **nine-palace grid with eight gates**; you read the layout and the on-duty gate.

> 開啟 / Open: 首頁選 **Qi Men Dun Jia · 奇門遁甲**。

![qimen 奇門局](img/qimen-chart.png)

## 怎麼讀 / Reading the board

```
 九宮（3×3）：八門安入八宮，中宮寄
 陽遁 / 陰遁 · 第幾局 · 值符門（開/休/生/傷/杜/景/死/驚）
```

- **陽遁 / 陰遁**：依節氣半年定順逆。
- **局數**：一到九局，定起盤方式。
- **值符門**：當下「當值」的門，**開門/休門/生門**為吉，**傷/死/驚**多凶。
